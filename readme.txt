Installation:

Macroscope requires the following libraries
rita wordnet www.rednoise.org/rita/wordnet/
toxiclibs code.google.com/p/toxiclibs/
proxml creativecomputing.cc/p5libs/proxml/

Download all these libraries and place them in the libraries folder inside your processing sketch folder. If the libraries folder doesn't exist yet, make one!

rename the unzipped folder to "macrosocope". 

that's it! Have fun
